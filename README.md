## 📖 О проекте

**Numerical Explorer** - это современное веб-приложение, разработанное в рамках курсовой работы на факультете компьютерных технологий и прикладной математики Кубанского государственного университета.

Приложение позволяет наглядно изучать поведение численных методов, сравнивать их эффективность и получать интеллектуальные рекомендации по выбору оптимального алгоритма для конкретной задачи.

---

## 🎯 Цели и задачи

- Изучение классических численных методов решения задач вычислительной математики
- Реализация 13 численных методов в едином программном комплексе
- Разработка системы интерактивной визуализации с возможностью масштабирования и перемещения графиков
- Создание модуля интеллектуальной рекомендации методов на основе анализа входных данных
- Проведение сравнительного анализа методов по ключевым метрикам

## 🚀 Возможности приложения

- ✅ **Интеллектуальная рекомендация** - анализ входной функции и выделение оптимального метода в выпадающем списке
- ✅ **Справка по методам** - краткое описание, условия применения и математическая формула при выборе метода
- ✅ **Сравнительный анализ** - запуск всех методов одновременно с выводом таблицы итераций, невязки и времени выполнения
- ✅ **Интерактивная визуализация** - построение графиков с возможностью масштабирования и перетаскивания
- ✅ **Динамические формы ввода** - адаптация полей под выбранный метод
- ✅ **Адаптивный интерфейс** - корректное отображение на всех устройствах

## 📊 Система интеллектуальной рекомендации

Приложение анализирует входные данные и автоматически рекомендует оптимальный метод:

### Уравнения
- **Гладкие функции, есть производная** → Метод Ньютона
- **Производная недоступна** → Метод секущих
- **Есть интервал со сменой знака** → Метод половинного деления
- **Функция с разрывами** → рекомендация недоступна

### Интегралы
- **Гладкие функции, высокая точность** → Метод Симпсона
- **Табличные данные или особенности** → Метод трапеций
- **Многомерные интегралы** → Метод Монте-Карло

### ОДУ
- **Высокая точность** → Метод Рунге-Кутты 4
- **Быстрая оценка** → Метод Эйлера

### СЛАУ
- **Малая размерность** → Метод Гаусса
- **Большая размерность, диагональное преобладание** → Метод Зейделя
